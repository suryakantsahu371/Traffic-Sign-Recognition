# 🚦 Traffic Sign Recognition System (TSR)

An intelligent **Traffic Sign Recognition (TSR)** system developed using multiple Machine Learning and Deep Learning models for accurate traffic sign classification and real-time detection.

---

## 📌 Project Overview

This project focuses on recognizing and classifying traffic signs using advanced AI techniques.  
The system was trained on the **GTSRB (German Traffic Sign Recognition Benchmark)** dataset and supports real-time traffic sign detection using OpenCV.

---

## ✨ Features

- 🚘 Developed TSR models using:
  - CNN (Convolutional Neural Network)
  - RNN (Recurrent Neural Network)
  - LSTM (Long Short-Term Memory)
  - KNN (K-Nearest Neighbors)

- 📂 Used the **GTSRB Dataset** for:
  - Traffic sign classification
  - Model training and testing

- 🖼️ Applied preprocessing techniques:
  - Image resizing
  - Feature extraction
  - Data normalization

- 📹 Implemented real-time traffic sign detection using **OpenCV**

- 📊 Evaluated model performance using:
  - Accuracy score
  - Prediction results
  - Classification performance
